
# xy-admin

## xy-admin是什么?
  一个基于vue2.x,vue cli4.0的一款后台管理项目。
  
## master分支和dev分支的区别?
    master分支，主要无数据请求，纯前端页面；
    dev分支，有node.js作为后台服务支持，需要配套的node代码，正在开发中...
